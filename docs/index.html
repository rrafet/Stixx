Stixx

Minimal, native sticky notes for macOS.

Download for macOS Source on GitHub

Free and open source. macOS 13 or later, Apple silicon and Intel. Under 1 MB.

Features

Quiet by default
No visible chrome. Controls appear only when you hover a note’s top edge.
Frosted glass
A translucent note blurs whatever sits behind it. A slider in Settings tunes how much color stays over the glass.
Lists and checklists
Type - for a bullet or [] for a checkbox. Click to check off, Tab to nest.
Nothing to save
Every note returns to its exact place on the next launch.
Quick capture
⌥⌘N creates a note from inside any app.
Search
⌘F finds any note, including the ones put away for later.
Tidy up
⌃⌘T aligns every open note into a calm grid.
Native and small
Pure AppKit, no dependencies, sandboxed. Notes never leave your Mac.

First launch. Stixx isn’t notarized yet. Right-click Stixx.app and choose Open the first time; if macOS still refuses, allow it under System Settings → Privacy & Security. Needed only once.
